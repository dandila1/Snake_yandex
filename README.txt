# Змейка
Это проект старая и знакомая всем игра змейка. Для запуска игры необходимо запустить файл main.
Программа состоит из 4 файлов: apple.py - описание класса яблоки (то, что съедает змейка)
			       main.py - основная программа 
			       setting.py - файл настроек
                               snake.py - описание класса змейки
в папке images хранятся все графические элементы - курсор, змейка, иконка, фоны игры
В файл result заносится результаты игроков.
